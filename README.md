# GODOT new project template
My personal godot initial project setup.

Folder Structure

- release   (optional) 
- addons    (optional)
- assets
    - Art
    - Sounds
    - Textures
    - Tiles
    - Themes
    - Fonts
- src
    - Actors
        - Actor.gd -  has shared signals and props eg: die, 
        - ActorSpawner.gd
        - Players
            - Player.tscn
            - Player.gd
            - Player.png
        - Enemies
            - ...
    - Objects
        - Pickables
            - Coin.tscn
            - Coin.gd
            - Coin.png
        - hazards
            - spikes
            - ...
